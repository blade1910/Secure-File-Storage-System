# Secure-File-Storage-System
A web-based cybersecurity project that provides secure file handling using hashing, encryption, integrity verification, and malware detection.
## Tech Stack
- Python
- Flask
- HTML/CSS
- Pandas
- PyCryptodome
- Requests
## Features
- File upload
- SHA-256 and SHA-512 hash generation
- Logging
- AES encryption/decryption
- Malware check using VirusTotal
